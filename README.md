# SchoolForum
本项目是一个基于 SpringBoot3 + Vue3 技术栈构建的前后端分离式校园综合服务平台，采用 JWT 校验方案实现身份认证与权限管理，致力于为高校师生打造一站式、高效率、高互动的数字化校园生活空间。
平台以 “连接校园、服务师生” 为核心，整合了三大核心场景：
校园交流场景：包含帖子广场、失物招领、校园活动、表白墙等功能模块，同时接入海文考研等合作机构入口，为师生提供信息共享、互助交流与资源对接的线上空间，打破传统校园信息壁垒。
教务服务场景：提供成绩查询、班级课程表、教务通知、在线图书馆、教室预约等功能，打通教务系统与校园服务的数据流，让师生足不出户即可完成教务办理与资源获取，大幅提升办事效率。
个人管理场景：支持个人信息设置与账号安全设置，通过完善的权限控制与数据加密机制，保障用户信息安全与使用体验，让用户安心享受平台服务。
在技术实现上，后端采用 MyBatis-Plus 简化数据持久化操作，通过 Redis 存储验证码并实现 IP 地址限流防刷，利用 RabbitMQ 异步处理短信发送任务，结合 SpringSecurity 与 JWT 构建严密的权限校验体系；前端则使用 Vue-Router 实现路由管理、Axios 处理异步请求、Element-Plus 构建交互友好的页面，并通过按需引入优化打包体积，适配深色模式提升用户体验。项目整体结构清晰、职责明确、注释全面，具备良好的可扩展性与可维护性，可有效支撑校园数字化服务的高效运转，助力高校打造便捷、智能、有温度的校园生态。
